Archived material should not be deleted unless explicitly requested. It can be reused as reference input, sample agents, fixtures, or historical context.

## Archive Decisions

| Path | Reason |
| --- | --- |

## Rule

Mainline project work belongs in the root AI Infra Lab platform. Domain apps and older strategy docs belong here unless they directly support the unified architecture.
